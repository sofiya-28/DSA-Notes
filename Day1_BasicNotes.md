# WHAY IS ALGORITHM? 
Algorithm--> A Step-by-step procedure to solve a problem.
Example-->
* Finding the largest number in array
* Sorting numbers in ascending order 
* Searching for an element

Real life Example--> Making Tea 
1. boil water.
2. Add tea leaves.
3. Add milk and sugar.
4. Serve tea.
--> These steps from an algorithm.
In programming , we also follow steps to solve problems.

 # SIMPLE EXAMPLE OF ALGORITHM , HOW ALGORITHM ARE WORK?
PROBLEM:--> Find the largest number in an array.
Array: 10 30 50 40 20
Algorithm: 
1. Assume the first number is largest.
2. Compare it with other number .
3. if a big number is found , update the largest number.
4. Print the largest number.
Answer: 50
